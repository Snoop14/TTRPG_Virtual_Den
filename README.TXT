PROJECT TITLE: TTRPG Virtual Den
PURPOSE OF PROJECT: Project
VERSION or DATE: March 2023
HOW TO START THIS PROJECT: Open files in Unity 2019, or load apk file onto head mounted display and then run it.
AUTHORS: Anoop S. Bains
USER INSTRUCTIONS: The user starts in front of the front door of the tavern. To the front of the user will be a board hanging from the 
		    ceiling with the basic instructions. These basic instructions include how to interact with objects in the scene as
		    well as how to teleport around the tavern. Many of the decorations in the scene are not interactable, but the 
		    lamps are. The main playing area is near the fireplace, in that area is a table with some objects laying on it, 
		    many of these objects are interactable, there is also a canvas nearby that can be used to add more objects to the scene.
